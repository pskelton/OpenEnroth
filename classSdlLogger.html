Logs provided message. The message will be ignored if the provided log level is lower than the log level set for the provided category. It is advised to also do the log level check at the call site to avoid message formatting overhead:

if (logger->LogLevel(ApplicationLog) <= LogDebug)
logger->Log(ApplicationLog, LogDebug, FormatMessage("blablabla %s %s", s1, s2).c_str());
